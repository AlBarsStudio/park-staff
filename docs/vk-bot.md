# 🤖 Бот ВКонтакте

ParkStaff имеет бота в беседе ВКонтакте, который позволяет  
управлять сменами без использования сайта.

---

## Подключение

Бот работает в **беседе ВКонтакте**.  
Для получения доступа к беседе обратитесь к администратору.

---

## Как работает бот
Сотрудник пишет команду в беседу
│
▼
Бот получает сообщение
│
▼
Идентификация сотрудника
по VK ID в таблице employees
│
▼
Запись данных в Supabase
│
▼
Бот отвечает в беседу

Бот напрямую взаимодействует с базой данных Supabase —  
данные, добавленные через бота, сразу отображаются на сайте.

---

## Команды бота

> ⚠️ Раздел будет дополнен после предоставления списка команд бота.  
> Для получения актуального списка команд напишите `/help` в беседу.

---

## Доступ

- Бот доступен **всем участникам беседы**
- Идентификация происходит по **VK ID** пользователя,  
  привязанному к профилю в таблице `employees` (поле `vk`)

---

## Связь бота с сайтом

| Действие в боте | Результат на сайте |
|---|---|
| Добавление смены | Смена появляется в разделе «Мои смены» |
| Удаление смены | Смена исчезает из календаря |
| Данные видны администратору | Да — в разделе «Управление сменами» |

---

*Документ будет дополнен командами бота.*
