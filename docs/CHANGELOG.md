# История изменений

Все значимые изменения проекта **ParkStaff** документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).  
Проект следует принципам [семантического версионирования](https://semver.org/lang/ru/).

---

## [2.3.0] — 2025-06-00

### Добавлено

- Полная документация проекта в папке `docs/`:
  - инструкция для сотрудника;
  - инструкция для администратора;
  - руководство по боту ВКонтакте;
  - описание структуры базы данных;
  - инструкция по развёртыванию.
- Файл `CHANGELOG.md` для ведения истории изменений.
- Файл `CONTRIBUTING.md` с правилами участия в разработке.
- Навигационная страница `docs/README.md`.
- Оптимизация количества запросов к базе данных.

### Изменено

- Обновлён `README.md` в корпоративном стиле с оглавлением и badge-ами.

---

## [2.2.0] — 2025-05-00

### Добавлено

- Бот ВКонтакте для управления сменами из беседы.
- Команды бота: добавление, удаление и просмотр смен.
- Поддержка диапазонов дат и неполных смен в боте.
- Маппинг VK ID на сотрудников системы.

### Изменено

- Доработана система ведения учёта и составления графика смен.

---

## [2.1.0] — 2025-04-00

### Добавлено

- Генератор графика смен с учётом приоритетов и доступности.
- Ручное составление графика.
- Просмотр графика в режимах день, неделя, месяц.
- Карточный и табличный режимы отображения графика.
- Фильтрация графика по аттракционам и сотрудникам.

### Изменено

- Улучшен адаптивный дизайн для мобильных устройств.

---

## [2.0.0] — 2025-04-00

### Добавлено

- Личный кабинет сотрудника:
  - календарь смен;
  - добавление и удаление смен;
  - просмотр графика от администратора;
  - отметка фактического времени работы;
  - расчёт заработной платы;
  - приоритеты аттракционов;
  - цель изучения.
- Панель администратора:
  - управление сменами;
  - управление аттракционами;
  - просмотр списка сотрудников;
  - настройка приоритетов.
- Журнал действий.
- Realtime-подписки для автоматического обновления данных.

### Изменено

- Полный переход на Supabase как единую платформу для базы данных, аутентификации и Realtime.

---

## [1.0.0] — 2025-03-00

### Добавлено

- Первоначальная версия проекта.
- Аутентификация через Supabase Auth.
- Базовая структура базы данных.
- Страница входа.
- Разделение ролей: сотрудник, администратор.

---

## Формат записей

### Типы изменений

| Тип | Описание |
|---|---|
| Добавлено | Новая функциональность |
| Изменено | Изменения в существующей функциональности |
| Устарело | Функциональность, которая будет удалена в будущем |
| Удалено | Удалённая функциональность |
| Исправлено | Исправление ошибок |
| Безопасность | Изменения, связанные с безопасностью |
