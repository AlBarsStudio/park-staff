# Документация ParkStaff

## О разделе

Здесь собрана полная документация по проекту **ParkStaff** — системе управления сменами для сотрудников и администрации парка аттракционов.

Документация предназначена для:

- сотрудников парка;
- администраторов;
- разработчиков и технических специалистов.
